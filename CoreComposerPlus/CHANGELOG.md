# Changelog

All notable changes to CoreComposerPlus will be documented in this file.

## [1.0.0] - 2026-01-07

### Added
- Initial release
- **Batch Processing** - Process multiple cores per cycle (configurable 1-100)
- **Speed Multiplier** - Adjust processing speed (0.1x to 10x)
- **Instant Mode** - Process all queued cores immediately
- **Enhanced Queue** - Memory Tree input can hold up to 8000 cores
- Debug logging option for troubleshooting
