# MobilityPlus

Adds advanced mobility equipment and traversal tools to Techtonica.

## Features

### Equipment
- **Stilts MKII** - Increased height over standard Stilts
- **Stilts MKIII** - Maximum height stilts
- **Speed Boots** - Move faster across the factory
- **Jump Pack** - Enhanced jumping capability
- **Rail Runner MKII** - Improved grappling hook for transit cables

### Placeable Items
- **Speed Pad** - Place down for a quick speed boost
- **Jump Pad** - Launch yourself into the air

## Requirements
- BepInEx 5.4.21+
- EquinoxsModUtils 6.1.3+
- EMUAdditions 2.0.0+
- TechtonicaFramework 1.0.0+

## Installation
Install via r2modman or manually place the DLL in your BepInEx/plugins folder.
