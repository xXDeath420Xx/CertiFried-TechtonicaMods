## Installation

1. Install [BepInEx 5.4.x](https://github.com/BepInEx/BepInEx/releases) for Techtonica
2. Install [EquinoxsModUtils (EMU)](https://thunderstore.io/c/techtonica/p/Equinox/EquinoxsModUtils/)
3. Install [EMUAdditions](https://thunderstore.io/c/techtonica/p/Equinox/EMUAdditions/)
4. Download this mod from [Thunderstore](https://thunderstore.io/c/techtonica/p/CertiFried/{MOD_NAME_NOSPACE}/)
5. Extract to `BepInEx/plugins/`

## Configuration

Configuration options can be found in `BepInEx/config/com.certifired.{MOD_ID}.cfg`
