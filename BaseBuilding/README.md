# BaseBuilding

Build base structures to expand and protect your factory.

## Features

- **Wall Panels** - Basic, reinforced, and heavy wall variants
- **Floor Tiles** - Metal flooring options including industrial and grated
- **Doors** - Standard and reinforced access doors
- **Corridors** - Connecting hallways and passages
- **Stairs** - Various stair configurations for vertical movement
- **Lighting** - Industrial and decorative light fixtures
- **Decorations** - Flags, signs, and aesthetic elements
- **Barriers** - Protective railings and energy barriers

## Crafting

All structures are crafted at the Assembler and require basic materials:
- Iron Ingots
- Limestone
- Steel Frames (for reinforced variants)
- Glass (for windows and lights)

## Installation

1. Install BepInEx
2. Install EquinoxsModUtils and EMUAdditions
3. Place BaseBuilding.dll in BepInEx/plugins

## Requirements

- BepInEx 5.4.21+
- EquinoxsModUtils
- EMUAdditions
