# CertiFried Modpack

The complete collection of CertiFried mods for Techtonica. This modpack brings together all officially released CertiFried mods in one easy installation.

## Included Mods

### Core Framework
- **TechtonicaFramework** - Core systems for health, damage, narrative, equipment, and environment mechanics

### Production & Machines
- **AdvancedMachines** - MKIII/MKIV/MKV Smelters, Assemblers, Drills, Threshers, and Planters
- **AtlantumReactor_Updated** - High-power Atlantum-based reactors
- **MorePlanters_Updated** - Additional planter variants
- **PlanterCoreClusters_Updated** - Clustered planter cores for efficient farming

### Logistics & Storage
- **EnhancedLogistics** - Improved conveyor and logistics systems
- **WormholeChests_Updated** - Linked storage chests
- **DroneLogistics** - Automated drone delivery system
- **Recycler** - Recycle items back into materials

### Exploration & Environment
- **AmbientLife** - Ambient creatures (butterflies, birds, fireflies, moths, dragonflies)
- **PetsCompanions** - Pet companions that follow you
- **HazardousWorld** - Environmental hazards and protective equipment

### Gameplay Systems
- **SurvivalElements** - Machine health and repair mechanics
- **MobilityPlus** - Enhanced movement equipment (stilts, speed boots, jump packs)
- **NarrativeExpansion** - Extended dialogue and quest systems
- **BioProcessing** - Biological processing chains

### Quality of Life
- **BeltImmunity_Updated** - Immunity to belt damage
- **ChainExplosives_Updated** - Chain reaction explosives
- **DevTools** - Developer tools for testing
- **OmniseekerPlus** - Enhanced scanning capabilities
- **Restock_Updated** - Auto-restock functionality
- **AtlantumEnrichment** - Atlantum enrichment processing

## Requirements

- BepInEx 5.4.21+
- EquinoxsModUtils 6.1.3+
- EMUAdditions 2.0.0+

## Installation

Install via r2modman or manually install all dependencies.
