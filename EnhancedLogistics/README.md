# Enhanced Logistics

Advanced logistics systems including universal search, storage networks, inserter enhancements, and drone delivery.

## Features

### Universal Search (Ctrl+F)
- Search Items, Recipes, Tech Tree, Machines
- Real-time search as you type
- Category tabs for filtered results

### Storage Network
- Network channel system for connected storage
- Auto-routing of items between networked chests
- Configurable distance limits (10-500m)

### Better Inserters
- Inserter range multiplier (1-3x)
- Inserter speed multiplier (1-3x)
- Smart filtering system

### Drone Delivery System (Press J)
- Drone spawning and management
- Configurable capacity (8-128 items)
- Configurable speed (5-50 units/second)
- Configurable range (50-500m)

## Requirements

- BepInEx 5.4.21+
- EquinoxsModUtils 6.1.3+
- EMUAdditions 2.0.0+

## Installation

Install via r2modman or manually place the DLL in your BepInEx/plugins folder.
