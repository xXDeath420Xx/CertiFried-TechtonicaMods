# Changelog

## v1.0.0
- Initial release
- Added 5 ambient creature types:
  - Butterflies (colorful, flutter near plants)
  - Birds (fly between perches)
  - Fireflies (glowing, darker areas)
  - Moths (nocturnal, attracted to light)
  - Dragonflies (fast, near water)
- Wandering AI behavior system
- Wing animation simulation
- Firefly light emission effects
- Distance-based spawn/despawn
- Configurable creature limits
