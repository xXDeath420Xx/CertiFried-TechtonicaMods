# DevTools

A comprehensive developer tools and cheats mod for Techtonica, exposing hidden game features and settings.

## Features

### Player Cheats
- **Infinite Crafting** - Build without consuming resources
- **Max Power** - All machines run at full power
- **Ultra Pickaxe** - Enhanced mining capability
- **Faster MOLE** - 4x MOLE speed
- **Disable Encumbrance** - No weight limits
- **Show Debug Coords** - Display debug coordinates
- **Hide Machine Lights/Particles** - Performance options
- **Sim Speed** - Adjust simulation speed
- **Free Camera Mode** - Photo mode

### Game Mode Settings
- **All Doors Unlocked** - Access all areas
- **Cat Sounds** - Toggle the cat sounds easter egg
- **MOLE Bits Always Available** - Always access MOLE drill bits
- **Infinite Ore** - Ore veins never deplete
- **Speed Multipliers** - Adjust player and MOLE speed percentages

### Machine Speed Multipliers
- Smelter speed percentage
- Assembler speed percentage
- Thresher speed percentage
- Planter speed percentage

## Configuration
All settings are configurable via BepInEx config file at:
`BepInEx/config/com.certifired.DevTools.cfg`

## Requirements
- BepInEx 5.4.2100+

## Installation
1. Install BepInEx
2. Place DevTools.dll in `BepInEx/plugins/`
3. Launch game and configure settings in the config file
