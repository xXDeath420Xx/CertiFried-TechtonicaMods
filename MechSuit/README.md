# MechSuit

Equip powerful mechanical exosuits for enhanced capabilities.

## Features

### Suit Variants

- **Standard Mech Suit** - Balanced stats for general use
- **Heavy Mech Suit** - Maximum protection and carrying capacity
- **Light Mech Suit** - Fast movement and agility focused

### Upgrade Modules

- **Power Core Module** - Increases suit energy capacity
- **Booster Module** - Enhanced movement speed
- **Armor Module** - Additional damage resistance
- **Shield Module** - Energy shield protection

## Crafting

Mech suits require advanced materials:
- Steel Frames
- Processor Units
- Mechanical Components
- Electrical Components

Modules require additional specialized components.

## Controls

- Suit abilities activated via configurable keybinds
- Energy management system with regeneration

## Installation

1. Install BepInEx
2. Install EquinoxsModUtils and EMUAdditions
3. Place MechSuit.dll in BepInEx/plugins

## Requirements

- BepInEx 5.4.21+
- EquinoxsModUtils
- EMUAdditions
